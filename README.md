## ARCHIVO .ENV
Antes que nada recuerda crear tu respectivo archivo .env con los siguientes datos:

### DATABASE CONNECTION
DB_HOST = localhost  
DB_PORT = 3306  
DB_NAME = dss  
DB_USER = root  
DB_PASSWORD = ""  

### DEEPSEEK API KEY
API_KEY = ""  
>[!IMPORTANT]
>Coloca en tu archivo .gitignore la carpeta del entorno virtual y el propio archivo .env.
